ECE 885 S20

Course Description

Over the past few years, Machine Learning (ML) has witnessed major advances in different areas. In applications that seemed to be impossible before, ML achieved remarkable performance (i.e., near human-level). These applications include healthcare, retail marketing, earthquake detection, machine translation, text to speech, object recognition and self-driving cars. However, the deployment of machine learning models to many real-world open the door wide to new kinds of cyber-security threats. From the eye of security and privacy, this course will introduce cutting-edge technologies for privacy-preserving AI. Furthermore, this course will introduce how to use ML to improve the security of a system and how to use ML to launch attacks and countermeasures.

Learning Outcomes

  1. Learn the cutting-edge technologies for privacy-preserving machine learning.

  2. Learn methods for measuring the data privacy.

  3. Learn about federated learning, a method for preserving data privacy by training models where data lives.

  4. Be able to define attacks on machine learning models and countermeasures.

  5. Learn methods for privacy-preserving encrypted evaluation of machine learning models.

  6. Explain how machine learning models can be used to launch attacks (Adversarial Learning) and counter security threats.

  7. Apply machine learning models to practice scenarios. Utilize programming.
